本ページは非公式の参考情報です。データは AWS Price List Bulk API の AmazonBedrock および AmazonBedrockFoundationModels オファーから GitHub Actions で毎日自動取得しています。 料金は予告なく変更されます。実際の請求は AWS 公式の料金ページおよび請求情報が優先されます。
